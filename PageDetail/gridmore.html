Project Results.

이번에 시작한 미니 프로젝트는 CSS Grid를 사용한 퍼블리싱 작업으로, 이미지를 스크롤하면서 인터랙션 옵저버를 통해 위치를 확인하고, 스크롤 시 이미지가 로딩되도록 구현했습니다. 또한 이미지에 호버 시 다운로드 버튼이 나타나도록 하여 사용자에게 인터랙티브한 경험을 제공했습니다. 처음 CSS Grid를 사용해 레이아웃을 구성하다 보니, CSS 코드가 자바스크립트처럼 복잡하게 느껴졌고, 이를 해결하기 위해 다양한 사이트와 예제를 참고하며 학습을 진행했습니다. 이미지도 하나하나 하드코딩하여 추가하며 작업을 진행했고, CSS Grid 연습뿐만 아니라 반응형 웹 디자인을 적용해 모바일과 태블릿, PC 환경 모두에 적합한 레이아웃을 만들었습니다. 이 프로젝트는 퍼블리싱 능력을 한층 향상시킨 좋은 경험이었습니다.
